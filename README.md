# Takumi Labs Homebrew Tap

Homebrew formulae for [Takumi Labs](https://takumilabs.com) projects.

## Install

```bash
brew tap takumitrader-netizen/tap
brew install mcp-shield
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| `mcp-shield` | Privacy-first security scanner for MCP servers |
